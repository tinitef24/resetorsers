# resetorsers
Resetorsers - невеличкий скрипт для платформи ArbitTerminal, який дозволяє зміни всі значення в полях "Max orders" на 0. Це особливо актуально, коли не можеш слідкувати за терміналом, і не хочеш щоб відкривались нові позиції, а також коли просто хочеш припенити відкриття нових будь-яких ордерів.
